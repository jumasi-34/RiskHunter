브라우저의 보안 정책(CORS)으로 인해 file:// 프로토콜을 사용해 직접 index.html 파일을 열 경우, 로컬 JSON 데이터 파일을 fetch() 할 수 없습니다.

🛠️ 시스템 해결 가이드 (추천)

  1. VS Code Live Server: index.html 파일을 우클릭하고 Open with Live Server (Go Live)를 선택하십시오.
  2. Python 로컬 서버: 현재 디렉토리에서 터미널을 열고 python -m http.server 8000를 구동한 후 http://localhost:8000로 접속하십시오.
  3. Node.js 로컬 서버: npx http-server ./ 명령을 터미널에서 구동하십시오.
