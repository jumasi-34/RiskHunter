# 💻 [Context 04_0] 전체 메뉴 통합 및 화면/업무 컨텍스트 개요 (Menu Workspace Integration)

본 문서는 정적 단일 웹페이지(SPA)로 구동되는 **Risk-Based Audit Checklist System (RiskHunter)**의 5대 핵심 사용자 메뉴, 전 화면 공통 플로팅 AI 비서, 그리고 관리자용 특수 보안 SQL 콘솔에 대한 화면 구성, 기능 구성 및 실제 현장 업무 시나리오(Workflows)를 통합 및 기획하는 전체 메뉴 통합 컨텍스트 개요서입니다.

---

## 🎨 대시보드 화면 및 UI 테마 디자인 핵심 가이드

본 시스템의 UI/UX 디자인은 사용자의 시각적 편의를 극대화하고 고품격 전문가용 플랫폼의 이미지를 심어주기 위해 다음과 같은 규칙에 따라 구현되었습니다:

*   **테마 및 색상 시스템 (HSL Palette)**:
    - **주조색**: 깊고 신뢰감을 주는 슬레이트 다크 블루 (Slate Dark Blue) 및 투명도가 절묘한 프리미엄 글래스모피즘(Glassmorphism Backdrops).
    - **강조색**: 전자기기 계기판 느낌의 테크니컬 시안 블루 (`hsl(192, 100%, 50%)`) 및 프리미엄 브론즈/웜 실버.
    - **리스크 경고색**: 리스크 수치에 따라 동적으로 노출되는 소프트 엠버(Yellow/Orange) 및 브라이트 오디트 레드 (`hsl(0, 85%, 60%)`).
*   **컴포넌트 레이아웃**:
    - **사이드바(Sidebar)**를 공통 컨트롤 타워로 삼아 활성 메뉴를 조정하고, 하단 프로필 스위처를 통해 권한별 화면(ADMIN 전용 메뉴 팝업)을 실시간 필터합니다.
    - **메인 영역**은 시각 지표(Top) ➔ 메인 콘텐츠 그리드 및 분석 패널(Middle) ➔ 세부 조치 가이드 및 리소스 인터랙션(Bottom)의 일관된 위계질서를 따릅니다.

## 📊 1. [메뉴 01] Dashboard (전체 Audit 현황 모니터링)

### ① 화면 레이아웃 (Layout)
*   **전사 Audit 현황 KPI 카드 영역 (Metrics Row)**:
    - 4개의 프리미엄 글래스모피즘 카드: **총 누적 리스크 건수 (Total Risks)**, **미해결 품질 이슈 (Open QI)**, **미점검 4M 변경점 (Pending 4M Changes)**, **과거 감사 미해결 지적사항 (Open Audit Findings)**이 실시간으로 상단에 표시됩니다.
*   **감사 준비 타임라인 및 조치 게이지 (Planning & Completion Rate)**:
    - 좌측: 예정된 Audit 일정과 준비 마감 디데이(D-Day)를 보여주는 타임라인 위젯.
    - 우측: 공장별 감사 지적사항 조치 완료율을 실시간 표현하는 원형/수평 게이지 바.
*   **공장별 리스크 누적 분석 차트**:
    - 공장별 실시간 리스크 가중치를 Plotly.js/Chart.js 기반으로 시각화한 수평 바 차트 탑재.

### ② 화면 기능 구성 (Features)
*   **글로벌 사이드바 필터 동기화**: 사이드바의 공장/공정 글로벌 필터 조작 시, 대시보드의 실시간 누적 점수와 차트 분포가 무지연(Zero-delay) 갱신됩니다.
*   **D-Day 경고 인터랙션**: 마감이 7일 이내로 다가온 예정 감사는 적색 경고 마커가 깜빡이는 모션 효과를 줍니다.

### ③ 사용자의 실무 활용 흐름 (Workflow)
```
대시보드 진입 ➔ KPI 카드로 전사 리스크 유효율 점검 ➔ 예정 감사 타임라인에서 다가오는 BMW Audit 확인 ➔ 취약 공정에 대한 선제 대비 필요성 인지
```

## 📅 2. [메뉴 02] Audit Planning (사전 일정 및 준비 항목 관리)

### ① 화면 레이아웃 (Layout)
*   **상단 일정 등록 및 타겟 셀렉터 (Registry Row)**:
    - **[신규 Audit 일정 등록]** 버튼 배치. 클릭 시 감사명, 대상 공장, 타겟 완성차 OEM, 감사 기간, 담당자 지정을 수행하는 모달 팝업이 노출됩니다.
*   **중단 기간별 준비 가이드 타임라인 (Preparation Feed)**:
    - 감사일 기준 준비 차수별(D-30: 규격 대조, D-14: 과거 QI 대책 분석, D-7: 4M 변경점 대조, D-1: 수검 증적 폴더 검검) 타임라인 안내 카드 피드.
*   **하단 사전 준비 필수 체크리스트 그리드 (Pre-Audit Readiness Checklist)**:
    - 감사 예정 공정의 의무 조항 준비 상태(`준비중` / `완료`)를 수동 토글할 수 있는 실시간 미니 테이블 제공.

### ② 화면 기능 구성 (Features)
*   **브라우저 로컬 저장소(localStorage) 영속성**: 모달을 통해 등록된 신규 일정과 사전 체크리스트의 토글 상태는 로컬스토리지에 즉시 기록되어, 페이지를 새로고침하더라도 유실 없이 보존됩니다.
*   **공장 필터 자동 연동**: 예정 감사가 등록된 공장을 선택하면, 해당 공장의 공정별 필수 감사 준비 지침이 최적화되어 자동 로딩됩니다.

### ③ 사용자의 실무 활용 흐름 (Workflow)
```
[신규 Audit 일정 등록] 클릭 ➔ 'DP공정 BMW 정기 Audit' 등록 ➔ D-14 및 D-7 준비 피드 확인 ➔ 하단 체크리스트에서 수검 증적 보완 여부를 완료(Checked)로 기록
```

## 📋 3. [메뉴 03] Plant Risk & Action (공장별 Risk 및 사후 조치 입력)

### ① 화면 레이아웃 (Layout)
*   **상단 공장별 실시간 감사 이력 타임라인 (Plant Audit History)**:
    - 선택된 공장의 과거 3개년 오디트 점수 추이 및 통과 등급 시각화.
*   **중단 공정별 리스크 원인 입체 매핑 (Risk Breakdown & Heatmap)**:
    - QI(품질이력), 4M(공정변경), FINDINGS(감사지적) 요소가 가류(Curing), 성형(Building) 등 15대 공정별로 기하학적으로 배치된 리스크 원인 비중 보드.
*   **하단 품질 지적사항 실시간 입력 및 상태 보드 (Corrective Action Board)**:
    - 과거 Audit 또는 이번 점검 중 검출된 세부 지적 사항 리스트 출력.
    - 우측 행에 **[조치 완료 완료(Close) 처리]** 버튼과 상태 뱃지(`Open` [적색] / `Closed` [녹색]) 배치.

### ② 화면 기능 구성 (Features)
*   **실시간 조치 상태 업데이트**: 조치 완료 클릭 시, 데이터 상태가 즉각 Closed로 변환되며, 해당 공장의 리스크 가중치가 실시간으로 감쇄 연산되어 차트와 대시보드 수치에 자동 반영됩니다.
*   **지적사항 추가 핸들러**: 현장에서 발견한 새로운 마찰 요인 및 부적합 사항을 즉석에서 리스트에 추가해 관리할 수 있습니다.

### ③ 사용자의 실무 활용 흐름 (Workflow)
```
MP 공정 선택 ➔ '가류기 압력 기록 오차' 지적사항(Open 상태) 발견 ➔ 현장 보완 실행 후 [조치 완료] 클릭 ➔ 즉시 Closed로 상태가 전환되며 공장 누적 리스크 점수 하락 확인
```

## 🔍 4. [메뉴 04] AI Action Advisor (지적사항 맞춤 대응 계획 도출)

### ① 화면 레이아웃 (Layout)
*   **상단 지적사항 검토 및 직접 입력 아레아 (Problem Input Card)**:
    - 외부 OEM 감사 또는 내부 점검에서 실제 지적받은 부적합 단어/상황을 사용자가 자연어로 직접 타이핑 입력하거나, 마스터 체크리스트의 지적 조항을 즉석에서 이관 선택할 수 있는 텍스트 아레아 제공.
*   **중단 AI 피드백 제안 트리거**:
    - **[AI 개선 조치 가이드라인 생성 (Generate AI Action Plan)]** 고성능 시안 블루 버튼 배치.
*   **하단 AI 시정 조치 피드 (AI Action Feed - Glassmorphism View)**:
    - 생성 클릭 시 우아하게 깜빡이는 진행 로딩바 작동 후, 전문 수석 오디터가 작성한 듯 정교한 **5대 가이드 라인**이 자동 출력됩니다:
      1. **Risk Summary** (위험 요약)
      2. **Root Cause Hypothesis** (근본 원인 가설)
      3. **Corrective Action (8D)** (영구 시정조치 계획)
      4. **Required Evidence** (합치 증빙 실물 가이드)
      5. **SOP Revision Guide** (연계 표준작업서 개정안 권고 및 교정 주기 룰)

### ② 화면 기능 구성 (Features)
*   **무결한 Mock-up 예외 설계 (SOP Mapper)**: 외부 API 연결 미비 상황에서도 "압력 검교정", "초물 검사", "혼련 온도" 등 사용자가 입력한 단어의 형태소를 분석하여, 실무 완성도가 극대화된 10여 가지 도메인 맞춤형 영구 개선 SOP 대책을 실시간 매핑 추론 출력합니다.
*   **인간-AI 대기 연출**: 생성 시 우아한 모던 스피너와 프로그레스 로딩 바가 연출되어, 실제 실시간 AI가 심층 품질 규격을 분석하는 듯한 경험을 제공합니다.

### ③ 사용자의 실무 활용 흐름 (Workflow)
```
'성형공정 그린타이어 보관 기준 시간 초과' 지적 내용 입력 ➔ [생성] 클릭 ➔ 로딩 후 AI가 제안하는 8D 시정조치안 및 '성형 SOP-04 내 보관 드럼 라벨링 보완 규칙' 수신 ➔ 대책서 즉각 반영
```

## 📁 5. [메뉴 05] Library (통합 감사 지식 라이브러리)

### ① 화면 레이아웃 (Layout)
*   **상단 통합 지식 서브 탭 컨트롤러 (Sub-Tabs)**:
    - **[Sub-Tab 1] Full Checklist Library**: 전사 공통 마스터 체크리스트 탐색 그리드. 우측 상단에 **[CSV 다운로드]** 단추 배치.
    - **[Sub-Tab 2] Customer Requirements Library**: BMW, Audi, Hyundai 등 완성차 OEM별 최신 규격 문서 리스트 및 AI 요약본 슬라이드인 패널.

### ② 화면 기능 구성 (Features)
*   **한글 깨짐 방지 UTF-8 BOM (UTF-8-SIG) 내보내기**: Full Checklist CSV Export 시, 한글이 깨지지 않고 엑셀에서 완벽히 인코딩되어 열리도록 BOM 마커 바인딩을 보장합니다.
*   **규격 요약 4대 지표 자동 추출 및 원본 파일 다운로드**: 특정 규격 문서 클릭 시 우측에 AI 규격 요약 피드가 팝업되며, `documents/` 디렉토리에 안착된 실물 PDF/DOC 파일 다운로드가 즉시 작동합니다.

### ③ 사용자의 실무 활용 흐름 (Workflow)
```
Library 탭 이동 ➔ [Customer Requirements] 서브탭 선택 ➔ Audi 가이드 문서 선택 ➔ 요약 패널 확인 및 [규격 원본 다운로드] 클릭 ➔ 로컬 저장소로 원본 파일 수신
```

## 💬 6. [전 화면 공통] Audit Assistant (플로팅 AI 챗봇 비서)

### ① 화면 레이아웃 (Layout)
*   **우측 하단 원형 플로팅 버블 (Floating Chat Trigger)**:
    - 챗봇 아이콘 형태의 은은한 글래스모피즘 원형 버블이 항상 화면 최상단에 배치됩니다.
*   **슬라이드 업 채팅 창 (Chat Drawer Backdrop)**:
    - 클릭 시 우측 하단에서 우아하고 매끄럽게 슬라이드인/업되는 채팅 대화방 컴포넌트 렌더링.

### ② 화면 기능 구성 (Features)
*   **화면 컨텍스트 감지형 첫 인사**: 사용자가 현재 `Audit Planning` 탭에 있는 상태에서 챗봇을 열면 **"BMW Audit 준비 일정을 관리 중이시군요. D-14 핵심 체크사항을 요약해 드릴까요?"**와 같이 현재 보고 있는 탭의 기획 컨텍스트를 완벽히 이해하고 대화를 리드합니다.
*   **자연어 질의응답 및 Mock 대화 피드**: 질문 입력 시 세련된 타이핑 애니메이션 효과와 함께 신뢰할 수 있는 전문적인 감사 가이드가 즉각 답변됩니다.

## ⚙️ 7. [ADMIN 전용] Admin Settings (통합 관리자 센터)

### ① 화면 레이아웃 (Layout)
*   **좌측 하단 프로필 계정 롤 연동**:
    - 일반 계정(`Viewer` 또는 `Manager`) 상태에서는 메뉴에 보이지 않으며, 프로필 스위처에서 **Lead Auditor (ADMIN)**를 선택하는 순간 사이드바 메뉴 최하단에 우아한 슬라이드 효과와 함께 Admin Settings 탭이 활성화됩니다.
*   **4대 통합 서브 탭**:
    1. **User Management**: 사용자 현황 요약 지표, 등록된 사용자 계정 리스트 그리드 및 퀵 사용자 전환(Simulate Session) 단추 제공.
    2. **Role & Permission**: 3대 역할별 권한 허용 한계를 나타내는 입체적인 권한 매트릭스 보드.
    3. **Audit Log**: 터미널 콘솔 감성의 감사 이력 실시간 로깅 피드 및 필터.
    4. **SQL Explorer**: 쿼리 템플릿 로더, 다중 라인 에디터 및 SELECT 전용 안전 샌드박스 경고 모달.

### ② 화면 기능 구성 (Features)
*   **실시간 변경 로깅 (Live Audit Trail)**: 시스템 전반에서 수행된 행동(일정 추가, 조치 종결, SQL 실행, 보안 경고 차단 등)이 발생할 때마다 실시간 로그에 영속 적재 출력됩니다.
*   **사용자 역할 동적 전환**: User Management 탭에서 특정 유저 프로필 전환 시, 전사 권한 상태가 변경되어 사이드바 및 UI 인터랙션 가드가 무지연 적용됩니다.
