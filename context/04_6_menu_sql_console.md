# 💻 04_6_menu_sql_console

본 문서는 **(ADMIN 전용) SQL Console** 화면의 구성 및 동작 요건을 정의합니다.

- **SELECT 템플릿 로더**: 자주 사용되는 원 데이터 탐색 쿼리 프리셋 제공.
- **안전 SELECT 샌드박스**: 파괴성 명령어(`INSERT`, `UPDATE`, `DELETE`, `DROP` 등) 입력 시 정규식을 통해 검사하고 차단 모달을 실시간 인서트.
- **가상 쿼리 프레임**: 클라이언트 메모리 단에서 마스터 JSON 자원을 매핑하여 결과를 동적 테이블로 시각화.
